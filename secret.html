You've discovered my secret page by clicking my profile picture 5 times! This page is a fun easter egg for visitors who like to explore and interact with websites.

How this secret page works:

This page is triggered by a JavaScript click counter that activates when you click my profile picture 5 times in succession.

// Secret page click detection
let profileClickCount = 0;
const profilePic = document.getElementById('profile-pic');

if (profilePic) {
    profilePic.addEventListener('click', () => {
        profileClickCount++;

        if (profileClickCount === 5) {
            window.location.href = 'secret.html';
            profileClickCount = 0;
        }

        // Reset counter after 3 seconds of inactivity
        setTimeout(() => {
            profileClickCount = 0;
        }, 3000);
    });
}
